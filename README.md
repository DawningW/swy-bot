# swy-bot
# 食物语自动挂机脚本

## 简介
食物语自动挂机脚本, 主要用于客潮和活动小游戏

有计划加入食物语做菜优化, 引用[此项目](https://github.com/ic30rs/swy_profit)

PS: 每次做菜都剩下一大堆食材好烦啊

## 食用方式
无

## 食材
- opencv-python
- pywin32
- PyAutoGUI
- pure-python-adb
- pyinstaller(如需要打包)
